# Simulated Tempering Langevin Monte Carlo (Ge, Lee, Risteski 2018)

- **arXiv**: 1710.02736 (NeurIPS 2018)
- **Key finding**: Provable polynomial-time mixing for multimodal Gaussian mixtures via simulated tempering with Langevin steps. Alternates Langevin at each temperature with temperature swaps.
- **Relevance**: Theoretical foundation for tempered Langevin. Shows geometric temperature spacing with swap acceptance ~0.234 is approximately optimal. Provides principled guidance on choosing number of temperature levels.
