# Beyond First-Order Tweedie: Solving Inverse Problems using Latent Diffusion
**Authors:** Rout, Chen, Kumar, Caramanis, Shakkottai, Chu
**Year:** 2024 | **Venue:** CVPR 2024
**Link:** https://arxiv.org/abs/2312.00852

## Key idea (1-2 sentences)
Second-order Tweedie approximation for latent diffusion inverse problems. Uses a surrogate loss to enable tractable Hessian trace estimation via random projections.

## Method summary (≤10 lines)
Extends PSLD (their NeurIPS 2023 work) with second-order Tweedie correction.
The key insight: first-order Tweedie (DPS-style) in latent space has quality-limiting bias.
Second-order correction adds the trace of the Hessian of the score function.
Uses Hutchinson's estimator for efficient trace computation via random projections.
Also includes a surrogate loss that avoids full Jacobian computation.

## Relevance to our problem
Closest prior work to Latent MMPS. However:
- STSL focuses on reconstruction quality (PSNR/SSIM), NOT calibration
- Uses Hessian trace (second-order score), while we use Tweedie covariance (first-order variance)
- Does not propagate V_t through decoder Jacobian — the key innovation of Latent MMPS
- No calibration metrics reported

## Key equations
Second-order Tweedie: E[x₀|x_t] ≈ x_t + σ²∇log p(x_t) + 0.5σ⁴∇²log p(x_t)
Surrogate loss: L(x_t) avoids explicit Jacobian of decoder

## Limitations noted by authors
Hessian trace estimation adds ~2× NFE cost. Limited to linear forward models in theory.

## Experimental takeaway
4-8× fewer NFEs than PSLD. Better PSNR/LPIPS than DPS/PSLD. No calibration analysis.
